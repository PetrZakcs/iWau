# Hra Reality

Příběhový systém výzev v reálném životě. Připoj se k pre-launch komunitě Hry Reality a buď mezi prvními 1000 hráči.

## Tech Stack
Tento projekt využívá moderní stack:
- Vite
- TypeScript
- React
- Tailwind CSS
- shadcn/ui

## Spuštění projektu lokálně

1. Nainstalujte závislosti:
   ```bash
   npm install
   ```
2. Spusťte vývojový server:
   ```bash
   npm run dev
   ```

## Sestavení pro produkci

1. Vytvořte produkční build:
   ```bash
   npm run build
   ```
2. Náhled produkční verze:
   ```bash
   npm run preview
   ```
